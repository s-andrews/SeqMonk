Strand Bias Quantitation

This method quantitates based on the directionality of your reads.

The equation used is (reverse-forward)/(reverse+forward) so you end up with a value between -1 and +1 depending on the bias in directionality.

Options

You can set a minimum count so that you don't add a lot of noise based on probes containing very few reads. Minimum count is based on the total of reverse + forward so reads with no associated strand are ignored.
